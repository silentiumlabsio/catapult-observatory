# Security Policy

## Scope

This repository is a public technical showcase for Catapult Observatory.

The complete private application source code is not included.

## Reporting a Security Issue

Do not publish suspected vulnerabilities, private device information, credentials, or sensitive evidence in a public GitHub issue.

Contact SilentiumLabs privately:

```text
Silentiumlabsio@gmail.com
```

Please include:

- A clear description
- Affected component
- Reproduction conditions
- Expected impact
- Sanitized supporting evidence

## Responsible Testing

Only test Catapult Observatory on hardware, devices, and networks you own or are explicitly authorized to assess.

## Supported Public Materials

The public repository may include:

- Documentation
- Architecture information
- Sanitized screenshots
- Demonstration results
- Hardware notes
- Technical limitations

It does not grant permission to access private SilentiumLabs infrastructure, source code, or systems.
